About Brick Breaker

Brick Breaker is a breakout clone which the player must smash a wall of bricks by deflecting a bouncing ball with a paddle. The paddle may move horizontally and is controlled with the System arrow Keys.

Read more >

Rules

1.Use only Left arrow key to move Left
2. Use only Right arrow key to move Right
3.The player gets 3 lives to start with.
4. A life is lost if the ball hits the bottom of the screen.
